### 6. 回测 vs 纸面交易策略一致性审计（重要发现）
- 报告：docs/audit-backtest-vs-paper-strategy-consistency.md
- 评分内核 + 成本算法一致 ✅
- **致命差异 1**：成交时点。回测 T-1信号+T日开盘成交（无前视）；纸面当日收盘价同时做信号+成交（前视偏差，高估纸面收益）
- **致命差异 2**：出场方式。回测清仓 vs 纸面 33% 分层减仓，盈亏曲线不可比
- 其他：仓位公式不一致、风控熔断两边都缺、配置来源双源
- 修复优先级：P0 成交时点 + P0 出场方式 → P1 仓位公式 + P1 风控熔断 → P2 配置单一化
