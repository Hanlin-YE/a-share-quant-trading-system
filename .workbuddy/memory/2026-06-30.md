### 1. LangAlpha token 节省策略调研
- 仓库：https://github.com/ginlix-ai/LangAlpha
- 核心机制：PTC (Programmatic Tool Calling) —— LLM 只写 Python 代码，通过 `execute_code` 在 Daytona 沙箱执行；原始金融数据用 MCP 工具在 Python 端处理（normalize/aggregate/downsample），只把精简结果回传 LLM
- 关键组件：`execute_code` + `generate_mcp_client_code()` + `data_client/{fmp,ginlix_data}` + provenance 中间件（`SNIPPET_MAX_CHARS` 截断）+ `downsampleBars` + memo cache + DeltaChannel 压缩
- 报告写入：`docs/langalpha-token-strategy-and-project-stack.md`
- 可借鉴点：本项目 `analyze_news_with_deepseek` 当前把 80 条新闻直接拼 prompt，可改 Python 端先 pandas 去重/聚类再喂摘要
